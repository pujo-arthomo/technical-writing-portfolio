# Smart Light Switch - User Manual

## Introduction
The SLW-100 Smart Light Switch is a Wi-Fi and Zigbee-enabled device that replaces a standard wall switch. It allows local control and remote operation via a mobile app.

This manual covers daily use, app setup, and basic features for end users.

**Safety Note**: Always turn off power at the circuit breaker before installation. This device must be installed by a qualified electrician if wiring is required.

## Key Features
- On/off control via touch or app
- Dimming support (for compatible bulbs)
- Scheduling and automation
- Voice control compatibility (Alexa, Google Home)
- Energy monitoring (approximate)

## Daily Operation
1. **Manual Control**: Touch the switch plate to turn lights on/off. Hold for 3 seconds to dim (if supported).
2. **App Control**: Use the companion app to control remotely, set schedules, or create scenes.
3. **Status Indicators**: LED ring shows connection status (solid blue = connected, flashing = pairing mode).

## App Setup (High-Level)
1. Download the app from App Store or Google Play.
2. Create an account.
3. Add device via "+" button and follow pairing instructions (Wi-Fi or Zigbee hub).

**Visual Reference**: The app home screen shows a switch icon with on/off toggle and dimmer slider.

## Limitations
- Requires stable 2.4 GHz Wi-Fi (5 GHz not supported)
- Zigbee mode needs a compatible hub
- No direct Bluetooth control

This device meets CE, FCC, and UL safety standards for residential use.
