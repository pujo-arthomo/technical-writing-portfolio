# Smart Light Switch - Quick-Start Guide

## Get Started in 5 Minutes
1. Install the switch (see Installation Guide or hire an electrician).
2. Download the app.
3. Turn on the switch — LED will flash for pairing mode.
4. In app: Add device → select Wi-Fi or Zigbee → follow prompts.
5. Name your switch and test remote control.

**Visual Reference**: Pairing screen shows a scanning animation with "Searching for SLW-100".

Success: LED solid blue.

Common Issue: If no pairing, reset by holding touch for 10 seconds.
