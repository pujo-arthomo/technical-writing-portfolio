# Device Operating Manual – CERTUS Flex Successor

This repository represents a long-form device operating manual created as a technical writing portfolio project.

The structure and content are designed to reflect a 200–300 page hardware and software manual developed using Adobe FrameMaker.

The documentation focuses on:
- clear device operation
- structured hardware and software explanations
- safety and compliance
- print-ready organization

All content is written as formal technical documentation suitable for regulated or industrial environments.
